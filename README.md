# Netflix-Data-analysis
This project analyzes netflix dataset to understands trends in movies and TV shows available on netflix.
The analysis explores content type , genres ,release trends ,and country wise distribution.
# Dataset
the Dataset contains information about Netflix titles including:
Title
Type(Movie or TV show)
Release Year
Country
Genre
Duration
Dataset Source: Netflix titles dataset.
# Tools Used:
Python
Numpy
Pandas
Matplotlib
jypiter Notebook
# Key Analysis Performed
Distribution of movies Vs TV shows
Most common genres on Netflix
Country wise content production
Content release trends over the years
# Visualizations
The project include multiple Visualizations such as:
Bar charts
the charts help in understanding Netflix content patterns and popularity trends
# Key Insights
Movies are more common on Netflix compared to TV shows.
Content Production increased significantly after 2015.
United states contributes the highest number os titles.
Drama and Comedy are among the most popular genres.
# Conclusion:
This analysis helps understand Netflix's content and trends over time.
Such Insights can help bussiness understand audience prefrences and streaming trends.
